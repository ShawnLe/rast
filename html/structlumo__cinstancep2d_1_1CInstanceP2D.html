void lumo_cinstancep2d::CInstanceP2D::generate ( )
virtual

generate a random test data i.e. model and image


1) model points are randomly created i.e.
–> m.p vec2(urand(-model_size, model_size), urand(-model_size, model_size))
–> m.a = urand(0.0, 2 * M_PI)
2) projected image points + noise are created
–> p.p = cmul(rotation, msources[i].p) + translation + randomUniformVectorFromCircle(error);
–> p.a = msources[i].a + angle + urand(-aerror, aerror);
3) cluttered image points are randomly created
