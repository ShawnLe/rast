void lumo_crasts2d::CRastS2D::start_match ( )
inline

invokes matching


used has size equals all image points; At the beginning, none image point is used
1) initially, parameter space for searching is [tlow, thigh] and all match pairs possible
2) queue (which is a heap).insert(state, ubound); here ubound is priority of that state
3) queue starts with 1 element i.e. initial state
4) algorithm 1 is applied:
–> pop top priority element & eval (i.e. calc upper bound, matches) it
–> remove state with top priority
–> check stop condition to terminate
–> if not terminate, split regions and eval them and put them to queue
